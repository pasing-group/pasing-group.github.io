- Custom cursor with lagging ring & page-wide glow aura
- Hero parallax + mouse depth effect
- 3D card tilt (mouse-tracked perspective)
- Magnetic buttons
- Scroll-driven reveal animations (staggered)
- Animated gradient name
- Counter animations on stats
- Tech tag wave entrance
- Full social links for Praveen & Pasing Group
